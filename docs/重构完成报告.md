## 重构方法

### 练习文件 (exercises/ 目录)
1. **移除答案实现** - 将具体实现替换为 `???` 或 `undefined`
2. **保留测试用例** - 所有 `expect` 断言保持不变
3. **添加提示信息** - 在关键位置添加注释提示
4. **保持结构完整** - 函数签名、注释和结构保持原样

### 答案文件 (solutions/ 目录)
1. **完整实现** - 包含所有函数的正确实现
2. **可运行测试** - 所有测试应该通过
3. **命名规范** - 文件名格式为 `{原文件名}.solution.js`
