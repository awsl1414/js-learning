## 使用方法

### 运行练习

```bash
# 运行指定练习
bun test exercises/02-运算符/01-运算符基础练习.js

# 运行所有测试
bun test
```

### 查看答案

参考答案位于 `solutions/` 目录下，与练习题文件一一对应：

```bash
# 查看答案文件
cat solutions/02-运算符/01-运算符基础练习.solution.js
```

## 练习格式说明

不同模块的练习题使用不同的占位符格式：

1. **02-运算符**: 使用 `???` 作为占位符
2. **03-控制流**: 完整示例代码
3. **05-数组操作**: 使用 `TODO` 和 `?` 占位符
4. **01-变量与类型**: 使用 `// ✏️` 注释标记

所有答案文件 (`.solution.js`) 都包含完整的实现代码。
