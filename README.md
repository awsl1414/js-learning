# JavaScript 学习项目

本项目是一个专门用于学习 JavaScript 编程语言的项目，使用 TypeScript 和现代 ES2025 标准。

## 安装依赖

```bash
bun install
```

## 运行项目

```bash
bun run index.ts
```

## 开发模式（热重载）

```bash
bun --hot index.ts
```

## 项目特点

- 使用 TypeScript 进行类型安全开发
- 遵循 ES2025 (ESNext) 最新标准
- 基于 Bun 快速 JavaScript 运行时
- 严格的 TypeScript 配置，确保代码质量
- 支持 JSX 和现代 JavaScript 特性

## 学习内容

- JavaScript 核心概念和语法
- TypeScript 类型系统
- 异步编程（Promise、async/await）
- ES6+ 模块系统
- 现代 JavaScript 特性
- DOM 操作和事件处理

本项目使用 `bun init` 在 bun v1.3.5 中创建。[Bun](https://bun.com) 是一个快速的 JavaScript 全栈运行时。
